# Gradient SwiftUI

基于 SwiftUI 3 和 iOS 15 的 feature 编写的基于Web Gradient 的 App，目前兼容了 iOS 版本和 watchOS 版本

该版本的 SwiftUI 比较完善，很多东西都可以很快的完成
